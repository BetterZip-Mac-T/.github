## Overview

BetterZip Advanced is an advanced archive management application for creating, browsing, and extracting archives in ZIP, RAR, 7z, TAR, GZ, BZ2, and dozens of other compressed formats with features that go beyond basic extraction into genuine archive management. It provides archive browsing without extraction, selective file extraction, archive updating, and compression with encryption that handle the complete lifecycle of working with compressed files.

Archive browsing displays the contents of any archive in a navigable file browser without extracting the entire archive, enabling inspection of file names, sizes, dates, and structure before committing to extraction. Quick Look integration previews individual files within archives without extracting them. The browsing capability saves time and disk space when only inspection or specific files are needed.

Archive creation compresses files into ZIP, 7z, TAR, GZ, and other formats with configurable compression levels, AES-256 encryption, and split archive creation for distribution. The compression handles large files and directory structures with progress tracking. Preset configurations save commonly used compression settings.

Selective extraction pulls specific files or folders from archives without extracting the entire contents, handling the common scenario of needing one file from a large archive. Drag-and-drop extraction works directly from the archive browser to any Finder location.

For users who work with archives regularly and need more capability than basic extraction — browsing without extracting, creating encrypted archives, selective extraction — BetterZip Advanced is the archive manager that handles every compression task.

## Performance

BetterZip balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
